# Texas Title Opinion Letter Template

## Usage Instructions

Use the docx skill to generate the Title Opinion Letter. This template provides the structure and content guidance.

**Before generating**:
- Complete all analysis steps in SKILL.md
- Have all findings organized
- Know the transaction type and client requirements

> ⚠️ **IMPORTANT**: All generated opinions are DRAFTS for informational purposes only. They do not constitute legal advice and must be reviewed, verified, and approved by a licensed Texas attorney before any use or reliance.
